# seu-unirobot

## Description

This project contains a controller, some debug tools and some scripts for seu-robocup kidszie team. The controller and debug tools are written in c++. The scripts are written in python.

## Build tools
* cmake >= 3.0
* gcc
* g++
* nvcc

## Dependencies

### c++ libraries
* cuda >= 9.0
* cudnn >= 7.0
* opencv >= 3.3.1
* libeigen3-dev
* libboost-all-dev
* freeglut3-dev
* libv4l-dev
* qt5-default
* astyle

### python libraries
* paramiko

## Build & Run
* cd path/to/project
* mkdir build
* cd build
* cmake ..
* make install
* cd ../bin
  
Then you can find all the executable files and data files

## Recommend OS
* ubuntu 16.04 64bit

## Recommend IDE
* Visual Studio Code(with c++ plugin)
* CLion
* Kdevelop >= 5.0
